## 简介

原版人口增长缺乏张力，战乱和疫病的破坏力偏低。本版本将粮食、战乱、瘟疫重新拉到核心位置：吃得好人口才涨，打得狠、病得重人口就会快速凋零。

## 核心特性

- **粮食绝对主导**：`cheap_food_in_location` 提升人口增长 +0.02，`expensive_food_in_location` 下降幅度 -0.012，缺粮与饥荒会明显压制增长。
- **战争惩罚更高**：劫掠、占领、召集征召兵都会带来持续的人口负增长，战争疲劳也会拖慢全国人口恢复。
- **瘟疫更致命**：黑死病死亡率抬到 50%~75%，天花、流感、麻疹的传播/致死参数上调，疾病将成为真正的人口杀手。
- **社会价值与政策**：部分价值观调整了粮食消耗、人口容量、征召规模，凸显战争与民生的取舍。

## 兼容性

支持 1.0.* 版本。主要改动人口/疾病/部分静态修正，对修改相同区域的其他 MOD 可能产生冲突。
