# 欧陆天际线

[github](https://github.com/yIllusionSky/Eurapo-Skyline)

欧陆风云5增强型平衡修改mod

## 简介

本mod旨在提升《欧陆风云5》的游戏体验。针对原版游戏中人口增长过慢的问题进行了优化，并全面增强了“种田”体验，包括优化人口机制、增强建筑系统、改进经济平衡等多个方面。

## 借用内容

本mod借用了以下mod的部分理念或内容：
- Better Hospital
- Megalopolis Upgrade
- 人口增长调整&季风

## 主要特性

- **人口机制优化** - 改进人口增长和损失的平衡性
- **建筑系统增强** - 新增和改进各类建筑物 (目前只更新了医院)
- **经济平衡调整** - 修改商品需求和建筑成本
- **常备军上限调整** - 原版常备军人数太少，死了都不心疼，此mod将常备军人数翻倍

## 核心机制改动

### 人口系统
- **人口增长与多重因素挂钩:**
  - **繁荣度 & 发展度:** 大幅提高人口增长率，高发展度的地区将迎来人口爆发。
  - **战争疲惫:** 会导致全局人口增长缓慢。
  - **荒废度 & 饥荒:** 将导致毁灭性的人口损失，恢复需要更长时间。
- **征召兵比例:**
  - 对多种征召兵的规模进行了小幅调整，以优化军事平衡。

## 包含内容

- 医院建筑改进(医院可以提高人口增长)
- 社会价值观系统优化(对部分较弱的价值观进行加强，这些价值观可以带来人口的提升)
- 商品需求平衡(提高建造城市的花fen)
- 新增都市(添加一种名为都市的建筑，参考`Megalopolis Upgrade`并用了该mod的素材，作者如果觉得不妥联系我)

## 兼容性

支持游戏版本：1.0.x 及以上
理论兼容除修改集镇，城市，乡村外(即修改了location_ranks)外的所有mod
